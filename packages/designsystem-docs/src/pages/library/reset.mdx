## Print styles

The following bootstrap print classes can be used directly to control the print preview:
.d-print-none, .d-print-inline, .d-print-inline-block, .d-print-block, .d-print-table, .d-print-table-row,  .d-print-table-cell,.d-print-flex, .d-print-inline-flex

In addition, the following custom print classes are available: (source code: designsystem/scss/print.scss, automatically imported in helsenorge.scss):
.col-print-{n}: creates a column based on bootstrap grid, where n is the column width (1/12) (corresponds to col-md-{n}/col-lg-{n)}
.order-print-{n}: defines the column order based on bootstrap grid, where n is the column width (1/12) (corresponds to order-md-{n}/order-lg-{n)}
.offset-print-{n}: defines the left column offset based on bootstrap grid, where n is the column width (1/12) (corresponds to offset-md-{n}/offset-lg-{n)}
