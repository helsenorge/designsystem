# Palette

<MDXCustom.Preamble>
  The color variables can be imported from the designsystem theme (js) or as sass variables.
</MDXCustom.Preamble>

## Usage

```
JS
import {theme as hndsTheme} from '@helsenorge/designsystem-react';
const mycolor = hndsTheme.palette[color];
```

```
SCSS
@import '~@helsenorge/designsystem-react/scss/_palette.scss';


```
