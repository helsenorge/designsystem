# Page-feedback

<Preamble>
  Page-feedback er et funksjonelt pattern som brukes til å hente inn tilbakemeldinger fra innbygger på om en siden
  fungerte som forventet
</Preamble>

![Bilde](Page-feedback-2col.png)

Page feedback er et kombinert pattern.

## Bruk

### Når skal Page-feedback brukes?

Page feedback brukes for å hente tilbakemeldinger fra innbygger på enkeltsider, av typen "Fant du det du lette etter?", "Var denne siden nyttig?", etc.

### Hvor bruke?

_Kan_ brukes på enkeltsider på Helsenorge.

### Hvordan bruke?

- Bare en instans av page-feedback pr side
- Page feedback skal vises nederst på siden etter innholdet men før footeren
- En eller flere knapper
- Knappene kan enten være tilstrekkelig feedback i seg selv, eller en eller flere knapper kan lede til ekspansjon av en Expanding-box med instruks om hvordan tilbakemelding ønskes oppgitt.
- Ved mottatt tilbakemelding vises en kvitteringsmelding, og knappene for tilbakemelding skjules.

## Virkemåte

### Bestanddeler

- Common-button, nivå 1
- Highlight, nivå 1
- H2

1. Normal (bare knapper og overskrift)
2. Aktivert (for innlegging av tilbakemelding)
3. Kvittering (kvitteringsmelding for mottatt tilbakemelding)
