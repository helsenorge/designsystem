# Tags

For å synliggjøre endringer på komponenter har vi innført tags. Tags er en visuell markering som viser ulike statuser for komponenter. Disse
vises til høyre for navnet til komponenten i sidemenyen og i toppmenyen inne på hver komponent sin dokumentasjonsside.

## Typer

Foreløpig er det disse tagsene vi har:

- **New**: Denne taggen indikerer at komponenten er ny eller har nye props/features.
- **Beta**: Ny komponent som ikke er stabil.
- **Breaking**: Komponenten har breaking changes i nyeste major-versjon.
- **Deprecated**: Komponent som er på vei ut av designsystemet.
- **Attention**: Komponent som har endringer som krever ekstra hensyn. F.eks. om det er nye UU-krav.
- **Code Only**: Komponent som ikke har en tilsvarende Figma-komponent.
