# Select

<MDXCustom.Preamble>
  Som innbygger vil jeg kunne se hva som er valgt av et liste med skjulte valgalternativer, og åpne den skjulte listen og gjøre andre valg
  dersom jeg ønsker det, slik at jeg kan løse mine oppgaver. Nytteverdi: Lar innbygger velge ETT blant flere alternativer. Bruksområde:
  Brukes når innbygger skal kunne velge ett enkelt valg under en etikett, men plassen som er til rådighet for kontrollen bare rommer ett
  valg.
</MDXCustom.Preamble>
